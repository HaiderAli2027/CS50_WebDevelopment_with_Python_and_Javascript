# What Selenium Does

With Selenium, you can:

- Open websites automatically
- Click buttons and links
- Fill out forms (login, search, etc.)
- Scrape data from web pages
- Run automated tests for websites

## How It Works

- Selenium sends commands to a browser
- A WebDriver (like ChromeDriver) acts as a bridge
- The browser performs actions (open page, click, etc.)

## How Selenium Helps in Web Testing

### Automates User Actions

Instead of manually testing, Selenium can:

- Click buttons
- Fill login forms

### Cross-Browser Testing

Websites may behave differently in:

- Chrome
- Firefox
- Edge

### Repeated Testing (Regression Testing)

Whenever developers update a website, old features can break.

Selenium can:

- Re-run the same tests again and again
- Ensure nothing is broken after updates

### Saves Time & Effort

Manual testing is slow and repetitive.

With Selenium:

- Tests run automatically
- Hundreds of test cases can run in minutes

### Real User Simulation

Unlike simple scripts, Selenium:

- Uses an actual browser
- Simulates real user behavior

So you catch real-world issues like:

- Buttons not clickable
- Slow loading elements
- Broken UI interactions

### Detects UI & Functional Bugs

Selenium helps find:

- Missing elements
- Wrong page navigation
- Broken links
- Form validation issues
